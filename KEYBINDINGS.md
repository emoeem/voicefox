# voicefox 键盘快捷键参考

## 全局快捷键

无论当前在哪个页面，以下快捷键始终有效：

### 导航

| 按键 | 功能 |
|------|------|
| `1` | 切换到**队列**页面 |
| `2` | 切换到**搜索**页面 |
| `3` | 切换到**排行榜**页面 |
| `4` | 切换到**收藏**页面 |
| `5` | 切换到**历史**页面 |
| `6` | 切换到**本地音乐**页面 |
| `7` | 切换到**设置**页面 |
| `/` | 切换到搜索页面 |
| `Tab` | 切换到下一个标签页 |
| `Shift+Tab` | 切换到上一个标签页 |

### 播放控制

| 按键 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `n` 或 `>` | 下一首 |
| `b` 或 `<` | 上一首 |
| `m` | 切换播放模式：列表循环 → 单曲循环 → 随机播放 |
| `]` | 快进 5 秒 |
| `[` | 后退 5 秒 |

### 音量控制

| 按键 | 功能 |
|------|------|
| `.` | 音量增加 5% |
| `,` | 音量减少 5% |

### 歌曲管理

| 按键 | 功能 |
|------|------|
| `Ctrl+L` | 收藏 / 取消收藏当前正在播放的歌曲 |

### 其他

| 按键 | 功能 |
|------|------|
| `q` | 退出 voicefox |
| `Esc` | 返回上一级或取消当前操作 |

## 队列页面（Main）

主页面显示正在播放信息、最近播放历史和歌词。

### 播放控制

| 按键 | 功能 |
|------|------|
| `←` | 后退 5 秒 |
| `→` | 快进 5 秒 |
| `↑` | 音量增加 5% |
| `↓` | 音量减少 5% |

> 注意：`←`/`→`/`↑`/`↓` 仅在队列页面中映射为播放控制。
> 在其他页面中方向键用于列表导航。

---

## 搜索页面（Search）

### 输入模式

按 `i` 或 `/` 进入搜索输入模式（如果已处于输入模式，直接输入文字）。

| 按键 | 功能 |
|------|------|
| 输入文字 | 自动触发搜索（300ms 防抖） |
| `Backspace` | 删除上一个字符 |
| `Enter` | 当输入框非空时手动触发搜索 |
| `Esc` | 退出输入模式 |

### 结果列表导航

| 按键 | 功能 |
|------|------|
| `↑` 或 `k` | 选择上一首歌曲 |
| `↓` 或 `j` | 选择下一首歌曲 |
| `Home` 或 `g` | 跳到列表顶部 |
| `End` 或 `G` 或 `Shift+G` | 跳到列表底部 |
| `PageUp` 或 `Ctrl+U` | 向上翻半页 |
| `PageDown` 或 `Ctrl+D` | 向下翻半页 |
| `Enter` | 播放当前选中的歌曲 |

### 音源控制

| 按键 | 功能 |
|------|------|
| `v` | 切换聚合搜索模式（同时搜索所有音源）/ 单音源模式 |
| `←` | 切换到上一个音源（仅在单音源模式下） |
| `→` | 切换到下一个音源（仅在单音源模式下） |
| `Esc` | 退出搜索页面，返回队列页面 |

### 鼠标操作

- 点击歌曲行 → 选中该歌曲
- 双击歌曲行 → 播放该歌曲
- 滚轮 → 上下滚动列表

## 排行榜页面（Leaderboard）

| 按键 | 功能 |

| `↑` 或 `k` | 选择上一个榜单 |

| `End` 或 `G` 或 `Shift+G` | 跳到底部 |
| `PageUp` 或 `Ctrl+U` | 向上翻半页 |
| `PageDown` 或 `Ctrl+D` | 向下翻半页 |

| 按键 | 功能 |
|------|------|
| `↑` 或 `k` | 选择上一首歌曲 |
| `↓` 或 `j` | 选择下一首歌曲 |
| `Enter` | 播放选中的歌曲（自动跳转到队列页面） |
| `←` 或 `Esc` | 返回榜单列表 |
| `Home` 或 `g` | 跳到顶部 |
| `End` 或 `G` 或 `Shift+G` | 跳到底部 |
| `PageUp` 或 `Ctrl+U` | 向上翻半页 |
| `PageDown` 或 `Ctrl+D` | 向下翻半页 |

### 鼠标操作

- 点击榜单行 → 进入该榜单
- 点击歌曲行 → 选中
- 双击歌曲行 → 播放

| 按键 | 功能 |
|------|------|
| `a` | 添加 JS 音源 URL |

| `↑` 或 `k` | 选择上一个音源 |
| `↓` 或 `j` | 选择下一个音源 |
| `s` | 切换到本地音乐目录区域 |

## 常见操作流程

### 搜索并播放歌曲

```
按 / 或 2    → 进入搜索页面
输入关键词     → 自动搜索
按 j/k 选择   → 选中歌曲
按 Enter     → 播放
```

### 收藏当前歌曲

```
在任何页面 → 按 Ctrl+L
收藏成功后会显示通知
```

### 切换播放模式

```
按 m → 依次切换：
  列表循环（默认） → 单曲循环 → 随机播放
```

### 切换音源

```
在搜索页面按 v   → 切换到单源模式
按 ← / →        → 切换音源
再按 v           → 回到聚合搜索
```

### 查看和管理收藏

```
按 4            → 进入收藏页
按 /            → 输入关键字筛选
按 d            → 取消收藏选中歌曲
```

### 自定义 JS 音源

```
按 6            → 进入设置页
选择"添加音源"   → 按 Enter
输入 JS 音源 URL → 按 Enter 确认
音源加载成功后自动启用
```
