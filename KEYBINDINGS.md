# voicefox 键盘快捷键参考

## 全局快捷键

无论当前在哪个页面，以下快捷键始终有效：

### 导航

| 按键 | 功能 |
|------|------|
| `1` | 切换到**队列**页面 |
| `2` | 切换到**搜索**页面 |
| `3` | 切换到**排行榜**页面 |
| `4` | 切换到**热门歌单**页面 |
| `5` | 切换到**收藏歌曲**页面 |
| `6` | 切换到**历史**页面 |
| `7` | 切换到**本地音乐**页面 |
| `8` | 切换到**设置**页面 |
| `/` | 切换到搜索页面 |
| `Tab` | 切换到下一个标签页 |
| `Shift+Tab` | 切换到上一个标签页 |

### 播放控制

| 按键 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `n` 或 `>` | 下一首 |
| `b` 或 `<` | 上一首 |
| `m` | 依次切换播放模式：列表循环 → 单曲循环 → 随机 → 顺序播放 → 播完停止 |
| `]` | 快进 5 秒 |
| `[` | 后退 5 秒 |

### 音量控制

| 按键 | 功能 |
|------|------|
| `.` | 音量增加 5% |
| `,` | 音量减少 5% |

### 歌曲管理

| 按键 | 功能 |
|------|------|
| `Ctrl+L` | 收藏 / 取消收藏当前歌曲；在歌单或收藏页作用于选中项 |

在搜索、排行榜歌曲、热门歌单歌曲、收藏、历史和本地音乐列表中，选中歌曲后：

| 按键 | 功能 |
|------|------|
| `a` | 将歌曲追加到播放队列末尾，不替换当前队列 |
| `Shift+A` | 将歌曲插入当前播放歌曲之后，作为下一首 |

如果当前队列为空，`Shift+A` 会将歌曲作为队列中的第一首；加入队列不会自动开始播放。

### 其他

| 按键 | 功能 |
|------|------|
| `q` | 退出 voicefox |
| `Esc` | 返回上一级或取消当前操作 |

### Windows 退出行为

- 按 `q` 会停止播放、清理终端界面并退出。
- 在 Windows Terminal 或 WezTerm 中直接关闭终端窗口时，voicefox 也会通过 Windows Job Object 自动结束 mpv，避免音乐继续播放或任务管理器残留 mpv 进程。

## 队列页面（Main）

主页面显示当前播放队列、歌曲信息和歌词。

### 队列操作

| 按键 | 功能 |
|------|------|
| `k` | 选择上一首队列歌曲 |
| `j` | 选择下一首队列歌曲 |
| `Home` 或 `g` | 跳到队列顶部 |
| `End` 或 `G` 或 `Shift+G` | 跳到队列底部 |
| `PageUp` 或 `Ctrl+U` | 向上翻半页 |
| `PageDown` 或 `Ctrl+D` | 向下翻半页 |
| `Enter` | 播放选中的队列歌曲 |
| `Shift+↑` 或 `Shift+K` | 将选中歌曲向前移动一位 |
| `Shift+↓` 或 `Shift+J` | 将选中歌曲向后移动一位 |
| `d` 或 `Delete` | 从队列移除选中的一首歌曲 |
| `Shift+D` | 清空整个播放队列 |

也可以按住鼠标左键拖动队列歌曲来调整顺序。

### 播放控制

| 按键 | 功能 |
|------|------|
| `←` | 后退 5 秒 |
| `→` | 快进 5 秒 |
| `↑` | 音量增加 5% |
| `↓` | 音量减少 5% |

> 注意：`←`/`→`/`↑`/`↓` 仅在队列页面中映射为播放控制。
> 在其他页面中方向键用于列表导航。

## 搜索页面（Search）

### 输入模式

按 `i` 或 `/` 进入搜索输入模式（如果已处于输入模式，直接输入文字）。

| 按键 | 功能 |
|------|------|
| 输入文字 | 自动触发搜索（300ms 防抖） |
| `Backspace` | 删除上一个字符 |
| `Enter` | 当输入框非空时手动触发搜索 |
| `Esc` | 退出输入模式 |

### 结果列表导航

| 按键 | 功能 |
|------|------|
| `↑` 或 `k` | 选择上一首歌曲 |
| `↓` 或 `j` | 选择下一首歌曲 |
| `Home` 或 `g` | 跳到列表顶部 |
| `End` 或 `G` 或 `Shift+G` | 跳到列表底部 |
| `PageUp` 或 `Ctrl+U` | 向上翻半页 |
| `PageDown` 或 `Ctrl+D` | 向下翻半页 |
| `Enter` 或 `l` | 播放当前选中的歌曲 |
| `a` / `A` | 加入队尾 / 插到当前歌曲之后 |

### 音源控制

| 按键 | 功能 |
|------|------|
| `v` | 打开当前歌曲的其他音源版本列表 |
| `←` | 切换到上一个搜索范围或音源 |
| `→` | 切换到下一个搜索范围或音源 |
| `Esc` | 退出搜索页面，返回队列页面 |

打开音源版本列表后：

| 按键 | 功能 |
|------|------|
| `↑` / `↓` 或 `k` / `j` | 选择其他音源版本 |
| `Enter` | 使用选中的音源版本播放 |
| `a` / `A` | 将选中的音源版本加入队尾 / 插到当前歌曲之后 |
| `Esc` 或 `v` | 关闭音源版本列表 |

### 鼠标操作

- 点击歌曲行 → 选中该歌曲
- 双击歌曲行 → 播放该歌曲
- 滚轮 → 上下滚动列表

## 热门歌单页面（Playlists）

顶部可在**已收藏、酷我、酷狗、QQ、网易云、咪咕**之间切换。在线音源的热门歌单会分别从对应音源加载。

| 按键 | 功能 |
|------|------|
| `←` / `→` 或 `[` / `]` | 切换上一个 / 下一个音源 |
| `↑` 或 `k` | 选择上一个歌单或歌曲 |
| `↓` 或 `j` | 选择下一个歌单或歌曲 |
| `Enter` 或 `l` | 进入歌单；在歌曲列表中播放选中歌曲 |
| `a` / `A` | 在歌曲列表中加入队尾 / 插到当前歌曲之后 |
| `←` / `h` / `Esc` | 从歌曲列表返回歌单列表 |
| `f` 或 `Ctrl+L` | 收藏 / 取消收藏当前歌单 |
| `r` | 刷新当前音源的热门歌单或当前歌单详情 |

### 鼠标操作

- 点击顶部音源 → 切换音源
- 双击歌单行 → 进入歌单
- 双击歌曲行 → 播放歌曲
- 右键歌单行 → 收藏 / 取消收藏

## 本地歌词与封面

- 本地歌曲优先读取与音频文件同名的 `.lrc` 文件。
- 没有 `.lrc` 文件时，会尝试读取音频内嵌歌词。
- 支持带时间戳的内嵌 LRC；纯文本内嵌歌词也会显示，并按歌曲时长分配显示时间。
- 封面只在队列主页显示，切换到本地音乐、历史等页面时会清理终端图片图层。

## 常见操作流程

### 搜索并播放歌曲

```
按 / 或 2    → 进入搜索页面
输入关键词     → 自动搜索
按 j/k 选择   → 选中歌曲
按 Enter     → 播放
```

### 收藏当前歌曲

```
在任何页面 → 按 Ctrl+L
收藏成功后会显示通知
```

### 切换播放模式

```
按 m → 依次切换：
  列表循环（默认） → 单曲循环 → 随机播放 → 顺序播放 → 播完停止
```

### 调整和删除队列歌曲

```
按 1                  → 进入队列页面
按 j / k              → 选择队列歌曲
按 Shift+J / Shift+K  → 将歌曲向后 / 向前移动
按 d                  → 从队列移除选中的一首歌曲
按 Shift+D            → 清空整个队列
```

### 删除本地音乐文件

```
按 7            → 进入本地音乐页面
按 j / k        → 选择歌曲
按 d / Delete   → 打开文件删除确认框
按 y            → 永久删除文件
按 n / Esc      → 取消删除
```

### 切换搜索音源

```
在搜索页面按 ← / → → 切换搜索范围或音源
选中歌曲后按 v      → 查看该歌曲的其他音源版本
在版本列表中按 Enter → 使用选中的音源播放
```

### 查看和管理收藏

```
按 5            → 进入收藏页
按 /            → 输入关键字筛选
按 d            → 取消收藏选中歌曲
```

### 浏览和收藏热门歌单

```
按 4            → 进入热门歌单页
按 ← / →        → 切换音源
按 Enter        → 查看歌单歌曲
按 f            → 收藏或取消收藏歌单
```

### 自定义 JS 音源

```
按 6            → 进入设置页
选择"添加音源"   → 按 Enter
输入 JS 音源 URL → 按 Enter 确认
音源加载成功后自动启用
```
